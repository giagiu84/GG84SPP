GG84 PWA iOS - cartella pronta per GitHub
Versione compatibilità: GG84_V.2_14.04.26

COME PUBBLICARE SU GITHUB PAGES
1. Crea un repository GitHub.
2. Carica tutti i file di questa cartella nella root del repository.
3. Vai su Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: main / root.
6. Apri l’URL GitHub Pages da Safari su iPhone.
7. Safari > Condividi > Aggiungi alla schermata Home.

NOTE TECNICHE
- Logica GG84 non modificata.
- Aggiunti manifest PWA, service worker, icone iOS, meta tag Apple, bootstrap PWA e CSS compatibilità iOS.
- GitHub Pages usa HTTPS: requisito necessario per Service Worker e WebCrypto.
- Su iOS l’app va installata da Safari con “Aggiungi alla schermata Home”.
